# LMS Frontend Scaffold

This is a scaffold for the Loan Management System frontend (Next.js App Router + TypeScript + Tailwind + shadcn/ui).

## Notes
- Put your existing shadcn UI components inside `src/components/ui/`
- Update `src/lib/api.ts` if backend response shapes change
- Add your actual page implementation inside the route folders already created

## Suggested route split
- Borrower app pages under `(borrower)`
- Employee/admin app pages under `(employee)`
- Public auth pages under `(public)`
